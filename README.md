# Simple Corona Map 

> Simple Map with React, Leaflet, Axios, Bootstrap

> Fetch API and shows actual Data


### Clone

- Clone this repo to your local machine using `https://github.com/StefanUp/CoronaMap`

### Setup

> npm install

> npm run start
